# 🌮 Tacos Gio Linh – Financial Decision Support System (DSS)

## 📖 Giới thiệu

Tacos Gio Linh DSS là hệ hỗ trợ ra quyết định tài chính được xây dựng trên nền tảng Microsoft Excel nhằm hỗ trợ đánh giá hiệu quả đầu tư, tối ưu hóa nguồn lực và quản trị rủi ro cho mô hình kinh doanh F&B.

Hệ thống kết hợp các phương pháp phân tích tài chính như **NPV**, **IRR**, **Solver** và **Data Table** để chuyển đổi dữ liệu kinh doanh thành các quyết định chiến lược.

## 🚀 Chức năng chính

* **Đánh giá khả thi đầu tư:** Tính toán NPV, IRR và phân tích dòng tiền dự án.
* **Tối ưu hóa nguồn lực:** Sử dụng Excel Solver để tìm cơ cấu sản phẩm tối đa hóa lợi nhuận.
* **Phân tích rủi ro:** Mô phỏng các kịch bản kinh doanh bằng What-If Analysis và Data Table.


## 🛠️ Công nghệ sử dụng

* Microsoft Excel
* Excel Solver (Simplex LP)
* What-If Analysis
* Conditional Formatting (Heatmap)
